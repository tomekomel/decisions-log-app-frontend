# decisions-log-app-frontend

Decision Log is micro app using Vue on frontend and NestJS with MongoDB on backend.


## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```
